# Instruções para Atualização do Repositório GitHub

## 📁 Arquivos Criados

Os seguintes arquivos foram criados para o novo site:

1. **index.html** - Página principal do site
2. **resumo_fluxos.html** - Página com o fluxo de marcação detalhado
3. **README.md** - Documentação do projeto

## 🔄 Como Atualizar o Repositório

### Opção 1: Substituição Completa (Recomendada)

1. **Acesse o repositório**: https://github.com/marcacaodse/distrito-eldorado-site
2. **Delete todos os arquivos existentes**:
   - Clique em cada arquivo (index.html, resumo_fluxos.html, etc.)
   - Clique no ícone da lixeira para deletar
   - Confirme a exclusão

3. **Faça upload dos novos arquivos**:
   - Clique em "Add file" > "Upload files"
   - Arraste os 3 arquivos novos (index.html, resumo_fluxos.html, README.md)
   - Adicione uma mensagem de commit: "Atualização completa do site - Nova versão"
   - Clique em "Commit changes"

### Opção 2: Criar Novo Repositório

1. **Crie um novo repositório**:
   - Vá para https://github.com/marcacaodse
   - Clique em "New repository"
   - Nome: `distrito-eldorado-site-2`
   - Marque como público
   - Clique em "Create repository"

2. **Faça upload dos arquivos**:
   - Clique em "uploading an existing file"
   - Arraste os 3 arquivos
   - Commit com mensagem: "Site inicial - Ferramentas Administrativas Distrito Eldorado"

## 🌐 Ativação do GitHub Pages

Para tornar o site acessível publicamente:

1. **Vá para Settings** do repositório
2. **Clique em "Pages"** no menu lateral
3. **Em "Source"**, selecione "Deploy from a branch"
4. **Em "Branch"**, selecione "main"
5. **Clique em "Save"**
6. **Aguarde alguns minutos** - o site estará disponível em:
   - `https://marcacaodse.github.io/distrito-eldorado-site/` (se usar o repositório existente)
   - `https://marcacaodse.github.io/distrito-eldorado-site-2/` (se criar novo repositório)
