# Ferramentas Administrativas - Distrito Sanitário Eldorado

Este é o site oficial das Ferramentas Administrativas do Distrito Sanitário Eldorado, desenvolvido para fornecer acesso público e gratuito a informações e recursos importantes para profissionais de saúde e usuários do sistema.

## 📋 Sobre o Site

O site foi desenvolvido com foco na organização clara de informações em categorias específicas, proporcionando acesso rápido e direto aos recursos necessários para o trabalho administrativo na área da saúde.

### Características Principais:
- **Design Moderno**: Interface limpa com botões de comando que levam diretamente aos links
- **Organização por Categorias**: Informações organizadas de forma clara e intuitiva
- **Acesso Público**: Disponível gratuitamente para todos os usuários
- **Responsivo**: Compatível com dispositivos móveis e desktop

## 🎨 Design

- **Cor de Fundo**: Branca
- **Títulos**: Azul escuro em negrito
- **Botões**: Azul escuro com texto branco em negrito
- **Tipografia**: Inter (Google Fonts)
- **Ícones**: Font Awesome 6.4.0
- **Framework CSS**: Tailwind CSS 2.2.19

## 🚀 Como Usar

1. Acesse a página inicial (index.html)
2. Navegue pelas categorias organizadas
3. Clique nos botões para acessar os recursos desejados
4. Use o botão "Tela Inicial" na página de Resumo dos Fluxos para retornar à página principal
